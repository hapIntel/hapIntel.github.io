1. jekyll is a static site generator.

常用的动态站点生成器：Wordpress， 


可以自动把markdown文件生成站点所需的html，css， js 文件。  

特点： 不用数据库 （简单）
       静态 （动态网站容易需要一个动态语言去支持： PHP，Java。 且动态网站容易被攻击。简单言之，静态网站就是一堆html语言。）
       GitHub page 兼容
       博客支持
